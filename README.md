## 📋 О проекте

**FusionTools** — создание полезных инструментов для Windows. Наша цель — объединить все необходимые программы и утилиты в одном месте.

---

## 🚀 Наши продукты

### 🦩 [FusionTools Store](https://github.com/devovv/FusionTools-Store)

Магазин приложений для Windows. Скачивайте популярные программы в один клик.

**Возможности:**
- 🏪 6 категорий, 35+ приложений
- 📚 Библиотека скачанных программ
- ⭐ Избранное — закрепляйте любимые программы
- 📦 Размер файла — видите объем перед скачиванием
- 🎨 Цвета акцента — 6 цветов на выбор
- 📥 Скачивание в один клик с прогресс-баром
- ➕ Добавление своих программ

[Подробнее →](https://github.com/devovv/FusionTools-Store)

## 📦 Установка

1. Перейдите в [раздел релизов](https://github.com/devovv/FusionTools/releases)
2. Скачайте `FusionToolsInstaller.exe`
3. Запустите установщик

---

## 🛠️ Системные требования

- Windows 10 / 11
- 50 MB свободного места
